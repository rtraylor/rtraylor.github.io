The Rigor of Fuzzy Sets https://www.themathcitadel.com/the-rigor-of-fuzzy-sets/ Wed, 21 Jun 2017 18:09:23 +0000 admin http://www.theresearchcortex.com/?p=763 Perhaps "fuzzy set theory", "fuzzy arithmetic", and "fuzzy rules" could have been named something a bit less mock-worthy. The word "fuzzy" has English synonyms "blurred", "unfocused", and the worst "ill-defined". However, fuzzy set theory is anything but fuzzy. Maybe cuddly and fun, but certainly not ill-defined. We'll take a look in this post at what exactly fuzzy sets are. Fuzzy set theory is an extension of classical set theory, so we'll first take a look at the basics of a classical set.  

Classical sets

A classical set is one we all deal with explicitly on a daily basis, even if we don't use that terminology. An item or element is either a member of a classical set, or it is not. Examples are
  • Even numbers, denoted [latex]2\mathbb{Z}[/latex]
  • A list of members of a particular country club
  • Wavelengths that are classified as UV light
  • The set of perfect squares [latex] S = \{ k^2 : k \in \mathbb{N}\}[/latex]
If we take an element in the relevant universe for each example, we are faced with the binary decision of whether or not it is a member of the set. 3 is not a member of [latex]2\mathbb{Z}[/latex]; 4 is. I am not a member of any country club, so given a particular one, I do not fall in that set. The electromagnetic spectrum is a continuum, but the cutoffs are strict. A wavelength between 10nm and 400 nm is ultraviolent. Any wavelength less than 10nm or greater than 400 nm is not. Finally, a the square of a rational number cannot be a perfect square, because a rational number is not a natural number. 3 is a natural number, but there is no natural number whose square gives 3. For all of these sets, we can define a characteristic function of a set [latex] A [/latex] (denoted [latex]\chi_{A}(x)[/latex]) that takes an input and returns a 1 if the input is an element of the set [latex] A [/latex] and 0 if it is not. Mathematically, we write [latex, display = true] \chi_{A}(x) = \left\{ \begin{array}{lr} 1, & x \in A \\ 0, &x \notin A \end{array}\right.[/latex] That is, if [latex]A = 2\mathbb{Z}[/latex], then [latex]\chi_{A}(3) = 0[/latex], and [latex]\chi_{A}(4) = 1[/latex]. Now, what if we relax the requirement that you either be in or out? Here or there? Yes or no? What if I allow "shades of grey" to use a colloquialism? Then we extend classical sets to fuzzy sets.

Partial Membership is well-defined

Many things we encounter daily aren't strict or crisp as is used in mathematics. Someone might tell you that you can either be in or out of a room, but what if you stand with one foot on either side of the threshold? It doesn't make sense to then say you are neither in nor out, because part of you is certainly inside the room. You can't "pick one", because the whole you is not wholly in either place all at once. Fuzzy sets allow for partial membership in a set. That is, you can be "a little bit" in the room (perhaps a toe is sticking in), "halfway" in the room (straddling the threshold), "most of the way" in the room (only a foot sticking out of the room), and everything in between. We formally define this notion by extending the characteristic function [latex]\chi_{A}(x)[/latex] to include the possibility of fractional membership. The membership function of a fuzzy set [latex]A[/latex] (denoted [latex]\mu_{A}(x)[/latex]) takes an element [latex]x[/latex] in a universe [latex] X[/latex] and maps it somewhere in the interval [0,1] according to the level of membership it has in the set [latex]A[/latex]. That is, [latex, display = true]\mu_{A}(x): X \to [0,1][/latex]   Example (Defining a bacon lover) Let [latex]x \in \mathbb{Q}[/latex] be the number of slices of bacon a person eats per day. (We will allow for partial consumption of bacon slices.) Suppose for simplicity that no one would ever eat more than 5 slices per day. Then a possible membership function for classifying someone as a "bacon-lover" could be [latex, display=true] \mu_{A}(x) = \left\{\begin{array}{lr}\frac{x-1}{2}, &1\leq x \leq 3 \\ 1, &3 \leq x\leq 5\\0, & \text{ otherwise}\end{array}\right.[/latex]   [caption id="attachment_775" align="aligncenter" width="522"] A possible membership function to classify as a bacon-lover[/caption] So given the number of bacon slices you eat per day, we can use the membership function [latex]\mu_{A}(x)[/latex] to determine your level of commitment to bacon. If you eat 2 slices of bacon per day, [latex]\mu_{A}(2) = 0.5[/latex], so you are a "halfway" bacon-lover.

 Level sets

We can also look at fuzzy sets in terms of their level sets, as a way to "backsolve" to determine what values of [latex]x[/latex] are at least an [latex]\alpha[/latex] level of membership in the set, where [latex]0\leq \alpha\leq 1[/latex].  Denoted [latex]A_{\alpha}[/latex], we define the [latex]\alpha[/latex]-level set as [latex, display=true]A_{\alpha} = \{x \in X : \mu_{A}(x) \geq \alpha\}[/latex] Example For the bacon set, we can use the graph above.
  • [latex]A_{1} = [3,5][/latex]
  • [latex] A_{1/2} = [2, 5][/latex]
  • [latex] A_{3/4} = [2.5,5][/latex]
In general, we can find the [latex]\alpha[/latex]-level set by inverting the membership function. In this case, the right endpoint is always 5, and the left endpoint is given by [latex]2\alpha + 1[/latex]. So, [latex]A_{1/8} = \left[ 2\cdot \frac{1}{8} + 1, 5\right] = [1.25 , 5][/latex] Some special level sets are the [latex]A_{1}[/latex], called the core, and [latex]A_{0}[/latex], called the support. The core tells us the interval that represents full membership, and the support tells us how "wide" the entire fuzzy set actually is. Now that we understand what fuzzy sets are, let's look at doing some things with them.

Fuzzy set operations are identical to classical set operations

We're going to dive a little further into fuzzy set theory and discuss operations on fuzzy sets such as union, intersection, and complementation. Since the fuzzy set is a nice extension of the classical set, the definitions of union, intersection, and complementation actually remain the same for fuzzy sets as for classical sets. The only difference is that the result is a fuzzy set rather than a classical set.[note] A classical set is really a special case of a fuzzy set.[/note] We'll discuss each one in turn.  

Union

The union is represented by the phrase "or"-- that is, given two sets [latex]A[/latex] and [latex]B[/latex], an element is a member of the union of [latex]A[/latex] and [latex]B[/latex] if it is in either [latex]A[/latex] or [latex]B[/latex].[note]"Or" should not be confused with "Xor" (exclusive or). 5 is in the union of odd numbers and prime numbers, and it is both an odd number and a prime number. That is, an element that is a member of both sets is also in the union.[/note] In classical set theory, the characteristic function of the union of two sets [latex]A[/latex] and [latex]B[/latex] is given by [latex, display = true]\chi_{A \cup B}(x) = \text{max}\left(\chi_{A}(x),\chi_{B}(x)\right)[/latex] Let's look at classical sets again, and let [latex]A[/latex] be the set of prime numbers, and [latex]B[/latex] be the set of odd numbers.
  • [latex]\chi_{A \cup B}(3) = \text{max}(A(3), B(3)) = \text{max}(1,1) = 1[/latex] so 3 is in [latex] A \cup B[/latex]
  • [latex]\chi_{A \cup B}(2) = \text{max}(A(2), B(2)) = \text{max}(1,0) = 1[/latex] so 2 is in  [latex] A \cup B[/latex]
  • [latex]\chi_{A \cup B}(4) = \text{max}(A(4), B(4)) = \text{max}(0,0) = 0[/latex] so 4 is not in  [latex] A \cup B[/latex]
For fuzzy sets, the formula for the membership function of the union of two fuzzy sets  [latex]A \vee B[/latex][note]We changed the notation here to show that we are not looking at classical sets anymore[/note] remains exactly the same as for the characteristic function of the union of two classical sets. [latex, display = true]\mu_{A \cup B}(x) = \text{max}\left(\mu_{A}(x),\mu_{B}(x)\right)[/latex] Let's keep our bacon set [latex]A[/latex] and define [latex]B[/latex] as the fuzzy set that classifies a "cheese-aficionado" with membership function [latex, display = true]\mu_{B}(x) = \left\{\begin{array}{lr}\frac{x}{5}, &0\leq x\leq5\\0, &\text{otherwise}\end{array}\right.[/latex] [caption id="attachment_785" align="aligncenter" width="560"] The black line is the bacon set A from before. The blue line is the new cheese set B[/caption]   We can now find [latex]\mu_{A \vee B}(x) = \max(\mu_{A}(x),\mu_{B}(x))[/latex]. For unions, the left endpoint of the support will be the smaller of the two left endpoints of the supports of [latex]A[/latex] and [latex]B[/latex], and the right endpoint of the union will be the larger of the two right endpoints of the supports of [latex]A[/latex] and [latex]B[/latex].  The support of "bacon-lover" union "cheese aficionado" will then be [0,5], which you can see from the graph. The membership function is given by [latex, display = true]\mu_{A \vee B}(x) = \left\{\begin{array}{lr}\frac{x}{5}, & 0\leq x \leq \frac{5}{3}\\\frac{x-1}{2}, & \frac{5}{3}\leq x \leq 3\\ 1, & 3\leq x \leq 5\end{array}\right.[/latex] [caption id="attachment_788" align="aligncenter" width="591"] The pink line gives the fuzzy set that is the union of bacon-lovers and cheese lovers[/caption]  

Intersection

The intersection of two sets denoted [latex] A \cap B[/latex] is represented in language by the word "and"-- an element must be a member of both sets simultaneously to be in the intersection of two sets. For the classical set example, we'll look again at [latex]A[/latex] as the set of prime numbers, and [latex]B[/latex] as the set of odd numbers. The membership function of the intersection of two classical sets, denoted [latex]\chi_{A \cap B}[/latex] is given by [latex, display = true]\chi_{A \cap B} = \min\left(\chi_{A}(x), \chi_{B}(x)\right)[/latex]
  • [latex]\chi_{A \cap B}(3) = \min(A(3), B(3)) = \min(1,1) = 1[/latex] so 3 is in [latex] A \cap B[/latex]
  • [latex]\chi_{A \cap B}(2) = \min(A(2), B(2)) = \min(1,0) = 0[/latex] so 2 is not in  [latex] A \cap B[/latex]
  • [latex]\chi_{A \cap B}(4) = \min(A(4), B(4)) = \min(0,0) = 0[/latex] so 4 is not in  [latex] A \cap B[/latex]
Just like for the union, the membership function of the intersection of two fuzzy sets (denoted [latex]A \wedge B[/latex]) has the same formula as that for the classical counterpart. [latex, display = true]\mu_{A \wedge B} = \min\left(\mu_{A}(x), \mu_{B}(x)\right)[/latex] The intersection of our fuzzy bacon set and our fuzzy cheese set then is given by [latex, display = true]\mu_{A \wedge B} = \left\{\begin{array}{lr}0, & 0\leq x \leq 1\\\frac{x-1}{2}, & 1\leq x \leq \frac{5}{3}\\\frac{x}{5}, & \frac{5}{3}\leq x \leq 5\end{array}\right.[/latex] [caption id="attachment_790" align="aligncenter" width="540"] The purple line gives the fuzzy set that is the intersection of the bacon set and the cheese set.[/caption]  

Complement

The complement of a set (denoted [latex]A^{c} \text{ or } \overline{A}[/latex]) is represented by the English word "not", and indicates negation. Everything outside a given set is the complement of a set. The membership function for the complement of a fuzzy set is identical to the characteristic function of the complement of a classical set: [latex, display = true]\mu_{A^{c}}(x) = 1-A(x)[/latex] The complement of a "bacon-lover", (a bacon-hater?) is then [latex, display = true]\mu_{A^{c}}(x) = \left\{\begin{array}{lr}\frac{3-x}{2}, & 1\leq x \leq 3 \\ 0, & 3\leq x \leq 5\end{array}\right.[/latex] [caption id="attachment_791" align="aligncenter" width="594"] The blue line is the complement of the bacon set[/caption]  

Conclusion

Fuzzy sets are simply an extension, or generalization of the classical sets we all implicitly knew already. A good definition, especially one that intends to extend a concept should contain the original concept as a special case. Fuzzy sets do indeed contain classical sets inside their definition. Moreover, operations on fuzzy sets should not need wholly new definitions. We saw that union, intersection, and complementation still hold for fuzzy sets just as they did before, so the theory of fuzzy sets is well defined and represents a consistent extension of classical set theory. The next post will extend the concept of fuzzy sets to fuzzy numbers, and then investigate how arithmetic works for fuzzy numbers.

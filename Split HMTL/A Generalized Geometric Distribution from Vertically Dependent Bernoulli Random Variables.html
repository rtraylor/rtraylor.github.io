Abstract

This paper generalizes the notion of the geometric distribution to allow for dependent Bernoulli trials generated from dependency generators as defined in Traylor and Hathcock's previous work. The generalized geometric distribution describes a random variable [latex]X[/latex] that counts the number of dependent Bernoulli trials until the first success. The main result of the paper is  [latex]X[/latex] can count dependent Bernoulli trials from any dependency structure and retain the same distribution. That is, if  [latex]X[/latex] counts Bernoulli trials with dependency generated by [latex]\alpha_{1} \in \mathcal{C}_{\delta}[/latex], and [latex]Y[/latex] counts Bernoulli trials with dependency generated by [latex]\alpha_{2} \in \mathscr{C}_{\delta}[/latex], then the distributions of  [latex]X[/latex] and [latex]Y[/latex] are the same, namely the generalized geometric distribution. Other characterizations and properties of the generalized geometric distribution are given, including the MGF, mean, variance, skew, and entropy.

Introduction

The standard geometric distribution counts one of two phenomena:
  1.  The count of i.i.d. Bernoulli trials until the first success
  2.  The count of i.i.d. Bernoulli trials that resulted in a failure prior to the first success
The latter case is simply a shifted version of the former. However, this distribution, in both forms, has limitations because it requires a sequence of independent and identically distributed Bernoulli trials. Korzeniowski [2] originally defined what is now known as first-kind (FK) dependent Bernoulli random variables, and gave a generalized binomial distribution that allowed for dependence among the Bernoulli trials. Traylor [4] extended the work of Korzeniowski into FK-dependent categorical random variables and derived a generalized multinomial distribution in a similar fashion. Traylor and Hathcock [5] extended the notion of dependence among categorical random variables to include other kinds of dependency besides FK dependence, such as sequential dependence. Their work created a class of vertical dependency structures generated by a set of functions [latex, display = true]\mathscr{C}_{\delta} = \{\alpha: \mathbb{N}_{\geq 2} \to \mathbb{N} : \alpha(n) < n \text{ and } \forall n \exists j \in \{1,\ldots,n-1\} : \alpha(n) = j\},[/latex] where the latter property is known as dependency continuity. In this paper, we derive a generalized geometric distribution from identically distributed but dependent Bernoulli random variables. The main result is that the pdf for the generalized geometric distribution is the same regardless of the dependency structure. That is, for any [latex]\alpha \in \mathscr{C}_{\delta}[/latex] that generates a sequence of identically distributed but dependent Bernoulli trials, the generalized geometric distribution remains unchanged. Background

The standard geometric distribution is built from a sequence of independent and identically distributed (i.i.d.) Bernoulli random variables with probability of success [latex]p[/latex] and probability of failure [latex]q=1-p[/latex]. There are two "versions" of the geometric distribution:
  • A random variable [latex]X[/latex] has a geometric distribution if it counts the number of Bernoulli trials needed to observe the first success.
  •  A random variable [latex]Y = X-1[/latex] has a geometric distribution if it counts the number of failures in a sequence of Bernoulli trials before the first observed success.
  In the first case, [latex]X[/latex] has support [latex]\{1,2,3,...\}[/latex], because we are looking for the first success, which can occur on trial 1, 2, 3, and so forth. In the latter case, [latex]Y[/latex] has support [latex]\{0,1,2,...\}[/latex] because we are counting the number of failures before the first success occurs. That is, if the first success occurs on Trial 1, then there were 0 failures preceding the first success. If the first success occurs on trial 2, then one failure occurred prior to the first success, and thus [latex]Y=1[/latex]. Essentially, Version 2 is a shifted Version 1, because our perspective changes-- we do not include the success in the count in Version 2. For Version 1, known as the standard geometric distribution the pdf is given by [latex, display = true]f_{X}(k) = q^{k-1}p, \quad k = 1, 2, 3, \ldots [/latex] For Version 2, (the shifted generalized geometric distribution)the pdf is given by [latex, display = true]f_{Y}(k) = q^{k}p, \quad k = 0, 1, 2, \ldots[/latex] The next section derives the generalized geometric distribution for FK-dependent random variables, and then shows that the pdf is the same regardless of dependency structure.

Generalized Geometric Distribution

Derivation from FK-Dependent Bernoulli Random Variables

Suppose we have a sequence of FK-dependent Bernoulli Random variables. Recall from [2] and [4] that FK-dependent random variables are weighted toward the outcome of the first variable [latex]\epsilon_{1}[/latex]. That is, in the Bernoulli case, [latex]P(\epsilon_{1} = 1) = p[/latex] and [latex]P(\epsilon_{1} = 0) = q = 1-p[/latex]. For subsequent variables in the sequence, [latex, display = true]\begin{aligned}P(\epsilon_{n} = 1 | \epsilon_{1}= 1) = p^{+} &\qquad P(\epsilon_{n} = 1 | \epsilon_{1} = 0) = p^{-} \\P(\epsilon_{n} = 0 | \epsilon_{1} = 1) = q^{-} &\qquad P(\epsilon_{n} = 0 | \epsilon_{1} = 0) = q^{+}\end{aligned}[/latex] for [latex]n\geq 2[/latex], where [latex]q = 1-p[/latex], [latex]p^{+} = p + \delta q[/latex], [latex]p^{-} = p-\delta p[/latex], [latex]q^{-}= q-\delta q[/latex], [latex]q^{+} = q + \delta p[/latex], and [latex]0 \leq \delta \leq 1[/latex] is the dependency coefficient. We will first give the generalized "Version 1" of the geometric distribution for FK-dependent random variables.  
Proposition. Suppose [latex]\epsilon = (\epsilon_{1},\epsilon_{2},\ldots, \epsilon_{n},\ldots)[/latex] is a FK-dependent sequence of Bernoulli random variables. Let [latex]X[/latex] be the count of such Bernoulli variables needed until the first success. Then [latex]X[/latex] has a generalized geometric distribution with pdf [latex, display = true]f_{X}(k) = \left\{\begin{array}{lr}p, & k = 1 \\q\left(q^{+}\right)^{k-2}p^{-}, & k \geq 2\end{array}\right.[/latex]
  In a similar fashion, suppose we prefer to count the number of failures before the first success occurs. For this generalized "Version 2", we have the following proposition.  
Proposition.  Suppose [latex]\epsilon = (\epsilon_{1},\epsilon_{2},\ldots,\epsilon_{n},\ldots)[/latex] is a FK-dependent sequence of Bernoulli random variables. Let [latex]Y = X-1[/latex] be the count of failures prior to the first success. Then [latex]Y[/latex] has a shifted generalized geometric distribution with pdf [latex, display = true]f_{Y}(k) = \left\{\begin{array}{lr}p, & k = 0 \\q\left(q^{+}\right)^{k-1}p^{-}, & k \geq 1\end{array}\right.[/latex]
 

Generalized Geometric Distribution for any Vertical Dependency Structure

Propositions 1 and 2 were derived for FK-dependent random variables, but in fact these random variables [latex]X[/latex] and [latex]Y[/latex] remain distributed according to the generalized geometric distribution and the shifted generalized geometric distribution regardless of the vertical dependency structure specified, as long as the dependency structure was generated from a function in [latex]\mathscr{C}_{\delta}[/latex].  
Theorem. Let [latex]\epsilon = (\epsilon_{1}, \epsilon_{2},\ldots,\epsilon_{n},\ldots)[/latex] be a vertically dependent sequence of Bernoulli random variables, where the dependency is generated by [latex]\alpha \in \mathscr{C}_{\delta}[/latex]. Let [latex]X[/latex] be the count of such Bernoulli trials needed until the first success, and let [latex]Y = X-1[/latex] be the count of failures of such Bernoulli trials prior to the first success. Then the pdf of [latex]X[/latex] and [latex]Y[/latex] is identical to those given in Propositions 1 and 2.
  This result is quite powerful, and not one that holds for all generalized distributions constructed from dependent random variables. Given any vertical dependency structure generated from the broad class [latex]\mathscr{C}_{\delta}[/latex], the count of trials before a success and the count of failures before a success have the same probability distribution. Thus, if this information is desired, no information about the dependency structure other than the membership of its generating function in [latex]\mathscr{C}_{\delta}[/latex] is necessary. The only information needed to calculate the generalized geometric probabilities for dependent Bernoulli trials is [latex]p[/latex] and [latex]\delta[/latex]. The next section gives some basic properties of the Generalized Geometric Distribution, such as the moment generating function and selected moments.   [/nextpage] [nextpage title="..."]

Properties  of the Generalized Geometric Distribution

Moment Generating Function

Fact.  The moment generating function of the generalized geometric distribution is [latex, display = true]M_{X}(t) = pe^{t} + \frac{qp^{-}e^{2t}}{1-q^{+}e^{t}}[/latex]

 

Mean

Fact.  The mean of the generalized geometric distribution is [latex, display = true]E[X] = \mu = \frac{1-\delta p}{p(1-\delta)}[/latex]
  The effect of dependence can be seen in the plot of [latex]E[X][/latex] below in Figure 1.. For fixed [latex]p[/latex], when [latex]\delta \to 1[/latex], [latex]E[X] \to \infty[/latex], though the rate changes with [latex]p[/latex]. [caption id="attachment_843" align="aligncenter" width="360"] Figure 1. Expectation of the Generalized Geometric Distribution.[/caption]   [caption id="attachment_845" align="aligncenter" width="447"] Figure 2. Expected value of generalized geometric distribution with fixed p=1/2[/caption] To explore further, suppose [latex]p = 1/2[/latex], and the Bernoulli trials are thus balanced between success and failure. Figure 2 shows the effect of delta for a fixed [latex]p[/latex]. Notice that the effect of [latex]\delta[/latex] on the expected value become more pronounced as [latex]\delta \to 1[/latex]. In particular, for [latex]\delta = 1/2[/latex] and [latex]p=1/2[/latex], [latex]E[X] = 3[/latex], but after this point, an increase of only [latex]1/6[/latex] in [latex]\delta[/latex] to [latex]\delta = 2/3[/latex] increased the expected value to 4 trials before a success. To double the expected number of trials before a success again to [latex]E[X] = 8[/latex] requires an increase of [latex]\delta[/latex] by only [latex]4/21[/latex] to [latex]6/7[/latex]. A smaller probability of success [latex]p[/latex] will yield an expected value [latex]\mu[/latex] that is much more susceptible to effects of dependency [latex]\delta[/latex], and a larger [latex]p[/latex] will yield an expected value more resistant to high dependency [latex]\delta[/latex]. Since the geometric distribution is a count of the number of trials needed to obtain the first success, a higher [latex]p[/latex] increases the probability that the first success occurs on the first trial, while a lower [latex]p[/latex] decreases that probability. Therefore, the dependency [latex]\delta[/latex] would have a higher effect for lower [latex]p[/latex], because a longer (dependent) sequence is expected to be generated prior to the first success, which increases the expected number of trials faster than if the Bernoulli trials were independent. Remark. Notice that when [latex]\delta = 0[/latex], the Bernoulli trials are independent. The mean of the generalized geometric distribution when [latex]\delta = 0[/latex] is [latex]E[X] = \frac{1}{p}[/latex], the mean of the standard geometric distribution.  

Variance

Fact.  The variance of the generalized geometric distribution is [latex display=true]\text{Var}(X) = \sigma^{2} = \frac{1-p + \delta p(1-p)}{p^2(1-\delta)^2}[/latex]
  [caption id="attachment_846" align="aligncenter" width="673"] Figure 3. Variance as a function of dependency for fixed p.[/caption] Figure 3 shows the effect of [latex]\delta[/latex] on the variance for different values of [latex]p[/latex]. As with the mean, a smaller [latex]p[/latex] induces a higher effect of [latex]\delta[/latex] on the variance of the number of dependent Bernoulli trials before the first success is observed. The shape of all 4 cases is similar, but the scales are vastly different. As [latex]p[/latex] increases, the scale of the variance decreases dramatically. Remark.  Again, note that when [latex]\delta = 0[/latex], the variance of the generalized geometric distribution reduces to that of the standard geometric distribution.  

Skew

Fact.  The skew of the generalized geometric distribution is given by [latex, display = true]\text{Skew}[X] = \frac{2-3p + p^2 + \delta p[q+\delta p q + p(2\delta-1-p)]}{\left(q + \delta pq\right)^{3/2}}[/latex]
[caption id="attachment_850" align="aligncenter" width="625"] Figure 4. Skew of Generalized Geometric Distribution as a function of p[/caption] The skew of the generalized geometric distribution gets more complicated in its behavior as a function of both [latex]p[/latex] and [latex]\delta[/latex]. Figure 4 shows the skew as a function of [latex]p[/latex] for the two extreme cases: complete independence ([latex]\delta = 0[/latex]) and complete dependence [latex]\delta = 1[/latex]. From [latex]p=0[/latex] to [latex]p \approx 0.658[/latex], the skew for the independent geometric distribution is greater than the completely dependent case. For [latex]p \gtrapprox 0.658[/latex], the skew is greater under complete dependence.  

Entropy

The entropy of a random variable measures the average information contained in the random variable. It can also be viewed as a measure of how unpredictable or "truly random" the variable is [1] . The definition of entropy, denoted [latex]H(X)[/latex], was coined by Claude Shannon [3] in 1948.   Definition. (Entropy) [latex, display = true]H(X) := -\sum_{i}P(x_{i})\log_{2}(P(x_{i}))[/latex]   For the standard geometric distribution, the entropy is given by [latex, display = true]H_{sg}(X) = \frac{-(1-p)\log_{2}(1-p)-p\log_{2}(p)}{p}[/latex]  
Fact. The entropy for the generalized geometric distribution is [latex, display =true]H_{gg}(X) = \frac{-\left[pp^{-}\log_{2}(p) + qp^{-}\log_{2}(qp^{-}) + qq^{+}\log_{2}(q^{+})\right]}{p^{-}}[/latex]

[caption id="attachment_852" align="aligncenter" width="463"] Figure 5b. Entropy as a function of dependency for fixed p[/caption]   Figure 5a shows [latex]H_{gg}(X)[/latex] as a function of [latex]p[/latex] for fixed values of [latex]\delta[/latex]. Notice that while the entropy decreases to 0 for all curves as [latex]p \to 1[/latex], the entropy curve is shifted upward for larger [latex]\delta[/latex]. Figure 5b fixes [latex]p[/latex] and looks at entropy as a function of [latex]\delta[/latex]. Notice that for smaller [latex]p[/latex], the entropy is much higher, which aligns with intuition. Conclusion

The standard geometric distribution counts the number of independent Bernoulli trials until the first success. This paper uses the works of Korzeniowski, and Traylor and Hathcock [2, 4, 5] on dependent Bernoulli and categorical random variables to develop a generalized geometric distribution built from dependent Bernoulli random variables. The main result of the paper is that the pdf for the generalized geometric distribution is independent of the dependency structure of the Bernoulli random variables that comprise it. That is, regardless of dependency structure, the pdf for the generalized geometric distribution is given by Proposition 1. Various properties and characterizations were given, including the moment generating function, mean, variance, skew, and entropy. The effect of dependency on each property was studied.
